Web Dev Explain

Web Dev Explain es un proyecto personal el cual tiene como principal finalidad registrar el proceso y evolucion del desarrollador del mismo asi como tambien conectar con sus demas proyector personales o proyectos utilizados en el portafolio, ademas de esto se tiene como segunda finalidad otorgar una pagina simple a aquellos desarrolladores web los cuales quieren una pagina para empezar o tener una guia muy cercana al usuario
